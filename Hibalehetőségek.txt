1. Az UPS oldal változékonysága, illetve pontos selectorok beállítása

ÖSSZESEN 5 helyen kell ellenőrizni a selectorokat:

- Tracking szám mező (beírás)
- Track gomb (kattintás)
- Proof of Delivery link (kattintás)
- Print this page link (kattintás)
- Save gomb (kattintás)

A. TRACKING SZÁM MEZŐ - ahol beírjuk a számot

# Tracking szám mező keresése
selectors = [
    (By.ID, "st_app_trackingnumber"),        # EZT KELL ELLENŐRIZNI
    (By.NAME, "trackingnumber"),              # EZT IS
    (By.CSS_SELECTOR, "input[placeholder*='Tracking']"),  # EZT IS
    (By.CSS_SELECTOR, "input[aria-label*='track']")       # EZT IS
]

B. TRACK GOMB - amire kattintunk a kereséshez

# Track gomb keresése
track_btn_selectors = [
    (By.ID, "st_app_track_package_btn"),      # EZT KELL ELLENŐRIZNI
    (By.XPATH, "//button[contains(text(),'Track')]"),    # EZT IS
    (By.XPATH, "//button[contains(text(),'Követés')]"),  # EZT IS
    (By.CSS_SELECTOR, "button[type='submit']")           # EZT IS
]

C. PROOF OF DELIVERY LINK - a POD oldal megnyitásához

# Proof of Delivery link keresése
pod_selectors = [
    (By.LINK_TEXT, "Proof of Delivery"),      # EZT KELL ELLENŐRIZNI
    (By.PARTIAL_LINK_TEXT, "Proof"),           # EZT IS
    (By.XPATH, "//a[contains(text(),'Proof')]"),         # EZT IS
    (By.CSS_SELECTOR, "a[href*='proof']")                # EZT IS
]

D. PRINT THIS PAGE LINK - a print nézet megnyitásához

# Print this page link keresése
print_selectors = [
    (By.LINK_TEXT, "Print this page"),         # EZT KELL ELLENŐRIZNI
    (By.PARTIAL_LINK_TEXT, "Print"),           # EZT IS
    (By.XPATH, "//a[contains(text(),'Print')]"),          # EZT IS
    (By.ID, "printLink")                        # EZT IS
]

E. SAVE GOMB - a PDF mentéséhez

# Save gomb keresése a print ablakban
save_selectors = [
    (By.ID, "save"),                           # EZT KELL ELLENŐRIZNI
    (By.XPATH, "//button[contains(text(),'Save')]"),      # EZT IS
    (By.XPATH, "//input[@value='Save']"),                  # EZT IS
    (By.CSS_SELECTOR, "button[onclick*='save']")           # EZT IS
]
